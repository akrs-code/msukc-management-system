A **full-stack club management system** developed for the **MSU Karate Club** to streamline membership registration, administrative approvals, and internal communication. This platform replaces manual processes with a secure, centralized, and data-driven solution.

## Overview

The MSU Karate Club Management System allows members to easily register and manage their accounts, while administrators gain full control over membership approvals, announcements, and activity monitoring. An intuitive admin dashboard provides visual insights to support informed decision-making and efficient club operations.

---

##  Features

### Member Features
- User registration and secure login
- Membership status tracking
- Access to club announcements

### Admin Features
- Membership approval and management system
- Admin dashboard with graphical insights
- Club announcements and updates
- Centralized member and activity monitoring

---

## Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT-based authentication  
- **UI & UX:** Responsive design with modern components
